Freestand Diagnostics — Support

Freestand Diagnostics is a companion app for Freestand training journal backups. Import a JSON export on your device to review summaries, tables, charts, and optional PDF reports.

How to get help

Choose any option below:

  • GitHub Issues (recommended)
    Report bugs, ask questions, or request features: github.com/ibeshesumne/Freestand-Diagnostics/issues
  • Open a new issue
    Use the New issue button on that page, describe your device and iOS/iPadOS version, what you tried, and what you expected.
